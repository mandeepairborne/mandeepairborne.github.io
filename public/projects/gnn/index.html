GNN-based Jet Classification Using Public LHC Data

Status: Ongoing
Project Type: Independent Research
Research Domain: High Energy Particle Physics, Machine Learning
Start Date: July 2025
Student: Mandeep Jangir

üéØ Objective

This project aims to explore the application of Graph Neural Networks (GNNs) for jet classification in high-energy particle physics. Specifically, the task involves distinguishing top-quark jets from QCD background jets using publicly available LHC datasets. The use of graph-based architectures provides a natural way to model the set of particles within a jet and the relational information between them.

üîç Motivation

Traditional methods for jet tagging, such as using Boosted Decision Trees (BDTs) or Convolutional Neural Networks (CNNs) on jet images, often lack the ability to fully capture the relational and sparse nature of jet constituents. GNNs offer a promising alternative by treating particles as nodes and interactions or kinematic proximity as edges in a graph structure. This approach aligns well with the irregular, hierarchical, and sparse structure of collider event data.

üß™ Dataset

  • Source: Deep Learning Top Tagging Challenge (available on Zenodo)
  • Contents: Simulated LHC proton-proton collision events
  • Labels: Top-quark jets vs. QCD background
  • Format: Each jet includes a variable-length list of particles with features such as:
    • Transverse momentum ( p_T )
    • Pseudorapidity ( \eta )
    • Azimuthal angle ( \phi )
    • Energy, charge, mass (optional)

üß† Methodology

Jet Graph Construction

  • Each jet is represented as a graph ( G = (V, E) ):
    • Nodes ( V ): Individual particles in the jet
    • Edges ( E ): Defined based on distance in ( \eta )-( \phi ) space or learned attention
  • Node features include ( p_T ), ( \eta ), ( \phi ), and other observables

Model Architecture

  • Baseline: EdgeConv and Message Passing Neural Network (MPNN) using PyTorch Geometric
  • Comparisons:
    • CNN-based jet image classifier
    • BDT with handcrafted observables (e.g., ( N )-subjettiness)

Evaluation Metrics

  • ROC AUC, classification accuracy
  • Precision-recall curve
  • Latent space visualization (e.g., t-SNE or UMAP)
